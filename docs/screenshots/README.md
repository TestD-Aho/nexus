# Nexus CMS Screenshots

This directory contains screenshots for the README.

## To add screenshots:

1. Take screenshots of the application
2. Save them here with these names:
   - `login.png` - Login page
   - `dashboard.png` - Admin dashboard
   - `page-editor.png` - Page editor with blocks
   - `media-library.png` - Media upload/gallery
   - `public-page.png` - Public page rendering

## Recommended screenshot sizes:
- Width: 1200px (max-width for README)
- Format: PNG
